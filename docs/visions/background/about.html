Citing¶

We encourage you to cite the package when using it for a publication. Visions was published in the Journal of Open Source Software. You can find the publication here. Please cite this work as:

Brugman et al., (2020). Visions: An Open-Source Library for Semantic Data. Journal of Open Source Software, 5(48), 2145, https://doi.org/10.21105/joss.02145
